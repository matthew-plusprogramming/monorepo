# Spec Schemas

This directory holds JSON Schemas for spec front matter and registry entries.

- `problem-brief.schema.json`: ProblemBrief front matter fields.
- `workstream-spec.schema.json`: WorkstreamSpec front matter fields.
- `master-spec.schema.json`: MasterSpec front matter fields.
- `contract-registry.schema.json`: Contract registry entry shape (id/type/path/owner/version).

Note: Required body sections are validated separately by spec tooling.
